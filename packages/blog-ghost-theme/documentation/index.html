C. How to Upload a Theme to Your Ghost Blog - top

1. Upload Through Admin

  1. Log into the admin section of your Ghost blog: yourblog.com/ghost.
  2. Click on General under settings in the left hand column and scroll to the very bottom.
  3. Click on Upload a theme.
  4. Drag-and-drop or click and locate the zip file for your theme.
  5. Once it is uploaded, click on either Activate Now to activate the theme immediately or Close if you want to activate later on.

2. Upload Manually

  • Locate the theme directory within the download files. It can be found at theme/forma.
  • Login to FTP/SFTP of your blog, or the file browser of your provided hosting service.
  • Navigate to the /content/theme directory within your Ghost installation.
  • Upload the forma directory found in step one into the remote /content/theme directory.
  • Restart your Ghost service. The process for this will change dependent on your provider. If you are unaware how to restart the service please raise a support issue with your host.
  • Log into the admin section of your Ghost blog: yourblog.com/ghost.
  • Click on General under settings in the left hand column and scroll to the very bottom.
  • Click on Activate next to the theme you want to activate and it will be activated.

D. Theme Features - top

1. Favicon

To add a favicon, navigate to General settings in your Ghost blog admin dashboard and click Upload Image next to Publication icon.

2. Navigation

You can add, edit, delete and reorder menu links on your Ghost blog, directly from the navigation tool within the blog admin area, located at /ghost/settings/navigation/.

To include a static page on your navigation menu, first type the name of the page as you’d like it to appear on your menu in the label field.

Next, click inside the URL field of the menu item. The blog URL will auto-populate with http://yourdomain.com/. You will need to add in the page slug after the final /. Once satisfied with your page link, click the blue Save button to add the page to the navigation menu.

3. Static Pages

In order to create a static page you create a new post, just like you would any other post. Once you have opened up the new post, there is a cog wheel icon next to where it says "Save Draft" or "Update Post" depending on if you have published the post or not. Click on that cog, and check the "Turn this post into a static page" box. This will convert your post to a static page.

4. Cover Images

You can add cover images to all of your posts and static pages. Open your post/page settings menu, drag and drop or locate the image on your computer, save it and you're all set.

5. Featured Content Carousel

The featured content carousel showcases up to five of your latest featured posts in a bold post carousel on the homepage. To mark a post as featured, open the post settings menu by clicking on the "gear" icon at the top right of the Ghost editor and check the "Feature this post" box.

Important! You need to enable Public API for the featured content carousel to work. To do this navigate to Labs in your Ghost admin dashboard and check the "Public API - For full instructions, read the developer guide" box.

6. Subscribers

To enable the "Subscribers" feature, navigate to Labs in your Ghost admin dashboard and tick the "Subscribers" checkbox. When the "Subscribers" feature in labs is enabled, the subscription form is displayed below the post content on the single post page and the optional Subscribe widget is displayed in the theme sidebar.

7. Image and Blockquotes Styles

When inserting images into your posts and pages, you can use special CSS classes to create a more attractive flow of content.

For instance, you can align your image left or right, by wrapping it in a div tag with the CSS class full-width, alignleft or alignright like this:

<div class="full-width">

<img src="/content/images/2017/12/my_image.jpg" alt="Alt Text">

</div>
<div class="alignleft">

<img src="/content/images/2017/12/my_image.jpg" alt="Alt Text">

</div>
<div class="alignright">

<img src="/content/images/2017/12/my_image.jpg" alt="Alt Text">

</div>

You can also add captions to your images. For this wrap your image in a figure tag with the CSS class caption and add your caption inside a figcaption tag with the CSS class caption-text like this:

<figure class="caption">

<img src="/content/images/2017/12/my_image.jpg" alt="Alt Text">

<figcaption class="caption-text">This is the caption</figcaption>
</figure>

If you want to add a caption and align an image left or right, use the following:

<figure class="caption alignleft">

<img src="/content/images/2017/12/my_image.jpg" alt="Alt Text">

<figcaption class="caption-text">This is the caption</figcaption>
</figure>

alignleft or alignright classes can be applied for blockquotes as well. For instance:

<blockquote class="alignleft">
In the beginner’s mind there are many possibilities, but in the expert’s mind there are few.
</blockquote>
<blockquote class="alignright">
In the beginner’s mind there are many possibilities, but in the expert’s mind there are few.
</blockquote>

8. Disqus Comments

Forma theme comes with Disqus commenting built-in, you just need to have a Disqus account (create it at Disqus.com). To enable commenting, you need to change the default Disqus account shortname "your_disqus_shortname" on line 17 in the default.hbs file located in the main theme folder (Ghost/content/themes/forma).

disqus_shortname = 'your_disqus_shortname', // add your Disqus shortname

If you wish to disable commenting, just remove the following lines from the post.hbs file located in the main theme folder:

{{!-- The tag below includes the theme comments - partials/comments.hbs --}}
{{> comments}}

9. Sidebar

The theme sidebar contains main site navigation, hardcoded text widget and Ghost newsleter subscription form. The sidebar.hbs file is located in the partials folder (Ghost/content/themes/forma/partials) of the theme.

Newsletter Subscription Form

If you have enabled the "Subscribers" feature in labs, Ghost subscription form widget is displayed in the sidebar.

If you want to integrate the MailChimp newsletter subscriptions, first, you need to generate the signup form code and add it istead.

Text Widget

You will find hardcoded text widget "About" in the sidebar of the theme.

To change the "About" widget information, edit the text on line 12 in the sidebar.hbs file located in the partials folder (Ghost/content/themes/forma/partials). If you do not need this widget, simply delete the section element:

<section class="widget widget-text">
  <h2 class="widget-title">About {{@blog.title}}</h2>
  <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Donec sed lacus id tortor iaculis iaculis dignissim eget justo. Maecenas posuere eros nisl, ut pharetra turpis faucibus a. In imperdiet urna mattis lacus mollis, sit amet pulvinar purus vehicula.</p>
</section><!-- .widget -->

10. Footer Widgets

The theme footer widgets contains "Latest Posts", "Tags" and "Instagram Feed" widgets. The footer-widgets.hbs file is located in the partials folder (Ghost/content/themes/forma/partials) of the theme.

Latest Posts Widget

The widget displays 3 latest posts by default, but you can change the number of posts on line 7 in the footer-widgets.hbs file located in the partials folder (Ghost/content/themes/forma/partials):

{{#get "posts" limit="3"}}

Important! You need to enable Public API for the widget to work. To do this navigate to Labs in your Ghost admin dashboard and check the "Public API - For full instructions, read the developer guide" box.

Tag Cloud Widget

The widget displays all blog tags ordered by name. You can change the order of tags or limit their number on line 27 in the footer-widgets.hbs file located in the partials folder (Ghost/content/themes/forma/partials):

{{#get "tags" limit="all" order="name asc"}}{{tags separator=""}}{{/get}}

Important! You need to enable Public API for the widget to work. To do this navigate to Labs in your Ghost admin dashboard and check the "Public API - For full instructions, read the developer guide" box.

Footer Instagram Feed

To display Instagram photos you need to change the default instagram_user_id and instagram_access_token on lines 18, 19 in the default.hbs file located in the main theme folder (Ghost/content/themes/forma).

instagram_user_id = 'your_instagram_user_id',
instagram_access_token = 'your_instagram_access_token',

To obtain an Access Token first you need to get a Client ID. Go to Instagram Developer page and log in. Click on Register your application. If this is your first application, register as developer by filling the information asked, accept the API terms of use and brand guidelines and click Sign up button. Click Register your Application again to proceed and then click Register a New Client at the top right. Set application name to your website name, type in a description, add the url of your site in the "Website URL" and "Valid redirect URIs" fields and enter your email. Click "Security" tab and uncheck the box "Disable implicit OAuth". Hit register.

Now, paste the following url into your web browser. Be sure to replace CLIENT-ID and REDIRECT-URI with your Client ID and Redirect URI you received when you registered your application in the previous step.

https://api.instagram.com/oauth/authorize/?client_id=CLIENT-ID&redirect_uri=REDIRECT-URI&response_type=token

Hit enter, and you will connect to Instagram's servers. Instagram will ask you whether you want to grant the client you registered access to your account. Once your client has been granted access to your account, Instagram will redirect you to your Redirect URI with the access_token in the url fragment. It will look like this:

http://your-redirect-uri#access_token=ACCESS-TOKEN

Simply grab the access_token off the URL fragment and add it to the default.hbs file.

To get your user ID paste the following url into your web browser. Be sure to replace ACCESS-TOKEN with your Access Token obtained previously.

https://api.instagram.com/v1/users/self/?access_token=ACCESS-TOKEN

If you want to remove copyright information in the footer, simply delete the p element with the class site-info and all code in it.

13. Google Analytics

First, you need to create a Google Account. If you already have one, you should be able to sign into Google Analytics using your existing Google Account credentials.

Once you've logged into Google Analytics, follow their guide to create an account for your blog. Once the account has been created, you'll receive a tracking code. Copy your tracking code and head over to your Ghost blog admin dashboard. Navigate to Code Injection settings and paste your tracking code into the Blog Header field.

Or you can paste your tracking code inside the <head> tag in the default.hbs file located in the main theme folder, right below the {{ghost_head}} and above </head>.

14. Search

The search functionality implemented in the theme requires Content API Key.

To obtain Content API Key, navigate to Settings → Integrations in your Ghost admin dashboard and click Add custom Integration button. Type the name of your custom integration and click Create. Copy your Content API key and click Save.

Next, open the default.hbs file located in the main theme folder (Ghost/content/themes/wildbird/) and replace your_search_key with your generated Content API Key on line 20:

search_key = 'your_search_key', // add your Content API key
